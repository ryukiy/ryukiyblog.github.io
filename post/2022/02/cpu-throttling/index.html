CPU のスロットリングにより Cubase の音が途切れてしまう

Posted by Ryuki Yoshimatsu on Thursday, February 17, 2022
Share:

僕の手持ちのラップトップで Cubase Pro 11 を立ち上げて、Cubase のデモトラック（Venus Theory Cubase 11 Demo Track）を再生すると、曲が盛り上がる部分で音が途切れてしまうという現象が起こります。原因を調べた結果、CPU のスロットリングによって起こっているのではないかという結論に至ったので、原因と対策について書いてみようと思います。

発生する現象

僕が使っているラップトップはマイクロソフトの Surface Book 2 で、以下のような構成のマシンです。

項目 スペック
CPU Intel(R) Core(TM) i7-8650U CPU @ 1.90GHz、2112 Mhz、4 個のコア、8 個のロジカル プロセッサ
メモリ 16.0 GB
SSD SAMSUNG MZFLW1T0HMLH-000MU、1TB

ラップトップのわりにはそこそこスペックも高いので Cubase の利用も問題ないだろうと思っていたのですが、デモトラックを再生してみると音が途切れてしまいます。一般的に、Cubase で音飛びする時にはオーディオインターフェースのサンプル数が低い場合に問題が起こることがあると思いますが、僕の環境では 512 Samples でも音が飛びます。

原因を調べていたところ、音が途切れるときには CPU のクロック数がグッと下がることが分かりました。Core i7-8650U にはクロック数をブーストする機能が搭載されているので、処理が重くなった際にはブーストして処理して欲しいところなのですが、実際は逆で、一番ブーストして欲しいところでクロック数が下がってしまうということが起こっているようです。

クロック数が下がる原因

クロック数が下がる原因をいろいろと調べていたところ、行きついたのが CPU のスロットリングです。

CPU に負荷がかかると、CPU の温度が高くなって不正な動作を行うことがあります。そこで、温度が高くなりすぎないように、あえて CPU の動作を遅くするための機能がスロットリングです。Cubase で音飛びする時には、このスロットリングが働いていることが分かりました。

スロットリングが発生していることを確認する方法ですが、ThrottleStop というツールを使って確認します。このツールは、スロットリングが発生している時には「POWER」という赤い文字を表示して、CPU の動作に制限がかかっていることを教えてくれます。僕の環境でデモトラックを再生すると、以下のような形でスロットリングが確認できました。曲を再生した直後は何度か POWER が点灯する程度で頑張っているのですが、曲の盛り上がり部分に入ったら POWER が点灯しっぱなしで音が止まってしまいました。

スロットリングの解決法

スロットリングによる Cubase の音飛びを解決するためには、スロットリングの機能を無効化します。ただ、スロットリングは CPU の温度が上がりすぎないようにするための安全対策のための機能なので、スロットリングを無効化するのはそれなりにリスクがあります。最悪の場合は、CPU の耐熱温度を超過して CPU 自体が壊れてしまうかもしれません。

パソコンが壊れるリスクを承知でスロットリングを無効化するには、ThrottleStop で [Disable and Lock Turbo Power Limits] にチェックを入れます。

ちなみに、Surface Book 2 ではこの解決策が使えませんでした。CPU の設定項目を変更するための機能が、Surface Book 2 では無効になっているためです。そのため、Surface Book 2 ではスロットリングを無効化することはできないのですが、安全対策という意味では確かに CPU の設定変更が出来ない方が安全といえるかもしれませんね。

結果的に、オーディオインターフェースのサンプル数を 1536 Samples まで上げたら音飛びが発生しなくなりました。かなり高い数値かなとは思いますが、一時的な対策としては使えそうです。
